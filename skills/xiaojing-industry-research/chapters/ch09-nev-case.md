# 第 9 章：案例：新能源汽车研究框架

## Core Idea
新能源汽车案例展示了如何把前 8 章行业研究框架串起来：先定位生命周期，再依次分析商业模式、市场规模、护城河、竞争格局、估值、外部因素和景气度。

## Frameworks Introduced
- **完整行业研究八步应用**：生命周期 → 可行性 → 规模性 → 防守性 → 盈利性 → 估值 → 外部因素 → 景气度。
  - When to use: 对一个行业做完整入门研究或写行业分析报告。
  - How: 每一步产出一个判断，再检查判断之间是否自洽。
- **新能源汽车生命周期判断**：用渗透率和销量增长判断行业仍处于成长期。
- **替代品护城河检查**：传统燃油车的资本、技术、供应链、客户关系是否能阻挡新能源汽车替代。
- **成长期估值宽容度**：若景气度强、交易不拥挤、价格战可控，可对估值更宽容；反之要谨慎。

## Key Concepts
- **新能源汽车产业链**：上游原材料，中游零部件和制造，下游整车、服务和软件应用。
- **换电模式**：新能源汽车中一个具有差异化的商业模式。
- **价格战**：成长期竞争加剧时常见，考验单车毛利率和资金实力。
- **月度销量**：新能源汽车行业高频景气指标之一。
- **市场导向**：行业从政策扶持逐步转向消费者和市场需求拉动。

## Mental Models
- **案例不是结论库，而是操作模板**：研究别的行业时复制的是分析顺序，不是新能源汽车的具体答案。
- **成长期看景气和竞争强度**：行业空间仍大时，短期投资判断常受销量、价格战和交易拥挤影响。
- **替代发生时，旧龙头护城河要重新评估**：传统优势可能被新技术路径绕开。

## Anti-patterns
- **只看销量不看利润**：销量增长可能伴随价格战和毛利下滑。
- **把政策驱动永久化**：行业成熟后，政策边际作用可能下降，市场需求变得更重要。
- **照搬案例结论到其他行业**：不同产业链、监管和技术成熟度下，框架应用结果会不同。

## Key Takeaways
1. 新能源汽车案例是前 8 章框架的完整演示。
2. 行业处于成长期时，市场规模、景气度和竞争强度很关键。
3. 商业模式可行性已验证后，重点转向空间、格局和估值。
4. 传统燃油车护城河不一定能抵御新能源汽车替代。
5. 月度销量是新能源汽车景气度跟踪的重要指标。
