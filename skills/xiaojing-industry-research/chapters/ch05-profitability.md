# 第 5 章：盈利性：行业的头部公司能分到多少钱

## Core Idea
成熟行业的关键不是市场还会不会高速增长，而是竞争格局决定谁能分到利润、分多少、能持续多久。

## Frameworks Introduced
- **横向格局 / 纵向格局**：竞争格局要从同行之间和产业链上下游两条线分析。
  - 横向格局: 同行业玩家如何分蛋糕。
  - 纵向格局: 上下游环节如何分蛋糕。
- **行业集中度分析**：集中度越高，龙头议价权和盈利能力通常越强。
  - When to use: 判断成熟行业能否出现稳定高利润龙头。
  - How: 看 CR3/CR4/CR5、龙头份额、集中度变化趋势。
- **产能周期四阶段**：供不应求 → 产能激增 → 产能出清 → 格局改善。
  - When to use: 周期品、制造业、资源品、资本开支驱动行业。
- **纵向议价框架**：利润分配取决于议价意愿和议价能力。
  - How: 看上下游集中度、供需紧张程度、替代选择、资金占用能力。

## Key Concepts
- **盈利性**：行业中企业尤其是头部企业获取利润的能力。
- **横向格局**：同行竞争结构，重点看集中度和产能周期。
- **纵向格局**：产业链利润分配结构，重点看议价权。
- **行业集中度**：头部玩家占市场份额的比例。
- **产能利用率**：实际产量与最大产能的比例，用于判断供需阶段。
- **议价能力**：交易对手是否有替代选择，以及你是否能影响价格和账期。

## Mental Models
- **成熟行业看“蛋糕怎么分”**：增长放缓后，赚钱能力取决于份额、壁垒和产业链位置。
- **产能周期解释利润波动**：扩产高峰往往压低利润，出清之后格局可能改善。
- **议价权来自没得选**：谁更稀缺、谁替代少、谁现金流更强，谁更容易分到利润。

## Anti-patterns
- **只看行业收入增长**：收入增长不等于利润增长，竞争恶化会吞掉增长。
- **忽略上游或下游**：公司份额高但被供应商或渠道压制，仍可能赚不到钱。
- **在产能顶部追周期**：高景气常引发扩产，随后可能供给过剩和利润下行。

## Key Takeaways
1. 盈利性要从横向同行竞争和纵向产业链分配同时分析。
2. 行业集中度上升通常增强龙头盈利能力。
3. 产能周期会导致行业利润周期性波动。
4. 纵向利润分配取决于议价意愿和议价能力。
5. 资金占用能力可以作为产业链地位的重要观察指标。
