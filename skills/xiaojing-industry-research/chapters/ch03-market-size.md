# 第 3 章：规模性：为什么有些公司上不了市

## Core Idea
成长期行业要重点判断市场规模，因为一个商业模式即使成立，如果市场太小，也很难孕育出值得投资或足够大的公司。

## Frameworks Introduced
- **TAM / SAM / SOM 三口径**：市场规模不是一个数字，而是不同问题对应不同口径。
  - TAM: 潜在市场，理论上总需求有多大。
  - SAM: 可服务市场，目前可被现实产品、渠道、区域服务的需求。
  - SOM: 可获得市场，某家公司实际能拿到的份额。
- **需求导向测算**：从用户数、频次、单价、渗透率、替代率推算需求。
  - When to use: 测算 TAM，尤其是需求端比较清晰的新兴行业。
- **供给导向测算**：从产能、资源、渠道、监管或核心瓶颈推算可实现规模。
  - When to use: 供给端限制明显，或公司实际可获得市场受瓶颈约束时。
- **供需匹配法**：在成熟行业中，通过“以小见大”或“以大见小”估算规模。
  - When to use: 供需存在稳定比例关系，或行业已相对饱和。

## Key Concepts
- **市场规模**：某一口径下需求或交易额的量化表达，必须说明口径和假设。
- **潜在市场 TAM**：理论上所有可能需求。
- **可服务市场 SAM**：在当前产品、渠道、价格、地区条件下可触达的需求。
- **可获得市场 SOM**：某家公司在竞争中可能获得的份额。
- **渗透率**：判断增长空间和生命周期阶段的关键指标。
- **替代率**：新产品替代旧产品或旧方案的比例。

## Mental Models
- **市场规模 = 口径 × 假设**：不要问“市场多大”，要问“以什么口径、基于哪些假设”。
- **小市场难出大公司**：即使商业模式好，市场天花板也会限制收入和估值。
- **测算不是算命，是暴露假设**：市场规模模型的价值在于把关键变量摆出来，方便验证和修正。

## Anti-patterns
- **混用 TAM、SAM、SOM**：用 TAM 讲公司收入潜力，容易夸大空间。
- **只引用券商或咨询报告数字**：不理解测算逻辑，就无法判断报告是否靠谱。
- **忽略供给瓶颈**：需求看似巨大，但产能、资源、渠道、监管可能限制实际规模。

## Key Takeaways
1. 成长期行业的核心问题是市场规模和增长空间。
2. 市场规模至少要区分 TAM、SAM、SOM。
3. 需求导向适合算潜在需求，供给导向适合看可实现规模。
4. 成熟行业可用供需匹配法做交叉验证。
5. 好的市场规模测算必须写清楚假设、变量和敏感性。
