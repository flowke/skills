# Memory Entry Template

用于 `docs-works/memory/*/*.md`，记录一条长期记忆。

适用场景：
- 已经通过 memory 筛选
- 需要沉淀为长期可复用文档

---

# <memory-title>

## 1. 结论
- 这条记忆的核心结论

## 2. 类型
- decision / constraint / pattern / fact

## 3. 适用范围
- 这条记忆适用于什么场景
- 不适用于什么场景（可选）

## 4. 依据
- 这条记忆来自哪个 discussion / delivery
- 是否已经验证

## 5. 内容
- 具体规则 / 事实 / 做法
- 保持简洁，优先写可直接复用的信息

## 6. 注意事项（可选）
- 使用时需要注意什么
- 常见误用点
