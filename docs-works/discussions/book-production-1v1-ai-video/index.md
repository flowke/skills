## 4. 全局关键待确认
- AI视频转换服务的唯一识别ID到底采用什么：题目ID、模型资源ID，还是内容生产侧生成资源ID？当前文档倾向“内容生产定义一个唯一的资源ID”，但需要确认。
- 视频上传后是否立即触发模型转换，还是由确定按钮 / 后台任务触发？文档中“调用模型接口”有删除线，需确认最新方案。
- 转换失败是否会发生、如何重试、重试上限、失败后是否允许人工重试或离线处理，需要确认。
- 素材库上传交互是否完全以 Figma 设计稿为准，产品文档中的未更新部分是否可以被 Figma 覆盖。
- AI视频资源、本地化资源、图书章节内容选择之间的互斥关系需要补全完整状态流。
- 图书扫码需求是否属于本轮内容生产平台范围，还是只作为后端消费链路背景。
