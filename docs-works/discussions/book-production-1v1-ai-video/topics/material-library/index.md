### R3. AI视频与 1v1 讲题模型转换
- 原始需求 / 原始描述：
  - 视频上传且题目ID填写完成后，点击确定。
  - 文档中“调用1v1讲题模型接口，将视频文件 + 题目ID发送至 1v1讲题模型服务”被删除线标记。
  - 需要以某个唯一识别ID连接内容生产后台与模型服务，并接收转换状态回调。
  - 生成状态包括：待转换、转换中、转换成功、转换失败。
  - 待转换：视频上传成功，尚未触发 / 调用接口。
  - 转换中：已调用大模型接口，等待回调。
  - 转换成功：视频资源转换成功，可进行图书绑定。
  - 转换失败：列表需要有失败状态。

  - 视频上传后是否立即触发模型转换，还是由确定按钮 / 后台任务触发。
  - 回调协议、状态枚举值、错误信息字段。
