# 图书管理 / AI视频资源绑定

- 当前状态：待确认

## 2. 需求地图

### R1. 学习内容新增 AI视频 类型
- 原始需求：在选择学习内容时，新增 “AI视频” 选项，点击展示当前图书目录下关联的所有 AI视频资源。
- 需求细化：
  - AI视频作为新的学习内容类型。
  - 选择 AI视频 后，展示当前图书目录下关联的 AI视频资源。
- 相关上下文：
  - C1 产品文档。
- 实现思路 / 方案预对齐：
  - 可复用现有学习内容选择弹窗 / 列表结构，新增 AI视频 tab 或类型选项。
  - AI视频资源列表的数据来源应与素材库转换成功资源或图书目录绑定关系关联。
- 待确认：
  - “当前图书目录下关联”的关联关系由哪个接口提供。
  - 仅展示转换成功资源，还是也展示待转换 / 失败资源。
- 当前结论：
  - AI视频应作为学习内容新增类型，但资源来源和过滤规则待确认。

### R2. AI视频列表字段
- 原始需求：AI视频列表字段包括视频ID、视频名称、题目ID、图书名称、创建者、资源ID。
- 需求细化：
  - 资源ID字段与素材库 topic 中的唯一资源ID策略相关。
  - 图书名称字段可能用于展示资源所属图书或关联图书。
- 相关上下文：
  - C1 产品文档。
- 实现思路 / 方案预对齐：
  - 列表字段应与素材库 AI视频资源实体保持一致，避免题目ID和资源ID语义分裂。
- 待确认：
  - 视频ID和资源ID的区别。
  - 图书名称是当前图书还是资源所属图书。
  - 是否需要分页、搜索、筛选。
- 当前结论：
  - 列表字段已列出，但字段语义依赖素材库资源模型确认。

### R3. 本地化和 AI视频资源互斥
- 原始需求：如果打开了本地化开关，则选择资源时不展示 AI视频列表；若已经在章节内容中选择了 AI视频资源，则外侧的本地化开关禁用。
- 需求细化：
  - 本地化开启时，AI视频资源不可选择 / 不展示。
  - 已选择 AI视频 资源时，外侧本地化开关禁用。
  - 两者形成互斥关系。
- 相关上下文：
  - C1 产品文档。
- 实现思路 / 方案预对齐：
  - 前端需要根据章节内容选择状态控制本地化开关 disabled 状态。
  - 选择资源弹窗需要根据本地化开关状态过滤 AI视频类型。
- 待确认：
  - 如果已经开启本地化，用户是否仍能看到 AI视频选项但 disabled，还是完全不展示。
  - 如果已选择 AI视频，是否允许删除 AI视频 后恢复本地化开关。
  - 保存时后端是否也校验互斥关系。
- 当前结论：
  - 互斥方向明确，但完整状态流和异常处理待确认。
